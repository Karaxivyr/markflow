# MarkFlow

MarkFlow is a small local-first markdown and bio formatter built with Vue 3, Vite, and SCSS.

## What it does

- Paste rough text into a clean editor
- Preview rendered markdown live
- Apply quick formatting from a toolbar
- Use preset templates for bios, project summaries, and Discord announcements
- Copy markdown to clipboard
- Export markdown as a `.md` file
- Autosave drafts locally with `localStorage`

## Stack

- Vue 3
- Vite
- SCSS
- markdown-it

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Notes

- No backend
- No auth
- No cloud sync
- No external CDN dependencies
- Core behavior runs fully on the user's machine
